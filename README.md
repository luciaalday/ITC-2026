# ITC-2026
## International Telemetry Conference Paper from the University of Arizona Theme Park Entertainment Group
This paper aims to provide research into an application of telemetry in theme parks and explore new ways telemetry can be used to enhance user experience.
By collecting information on how force from users can be generated into usable energy and displaying this information in a gamified format, we aim to explore the aspects of sustainability and investigate the efficacy of different force to energy storage systems. By conducting the experiment on a small scale using Arduino sensors and parts, we will be testing with real data and investigate how this would scale into larger, real-life applications
